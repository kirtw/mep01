Caller methodology

        D Init^hgh()      Caller must init, set timing of cumulation...
        D HGS^hgh       TI(), TIVL       creates array HTS()   buffered, start of page, head, style if in head
            D guts

        D HGE^hgh      Creates HTE()  close of tags
        D WH^hgh(devHW)    ; actually write HTS,HT,HTE  with hcss  to html file

guts consuists of successive calls  to
        sv^hgh(h)   where h is a piece of html or just a text string
        ot^hgh(tgc)    then inside   then D ct^hgh(tgc)   same tgc to close it
            tgc  is usu literal  ".xyz"   pfx dot => class
                        "#abc"  is id value, must be unique for whole page
                        "div"  a specific tag  div is implied with above class or id
                        "article"   eg. for content articles scraped
            values match hcss to acquire those properties
        sp^hgh(n)  produces n &nbsp;  non-breaking space chars, encoded
       

 css is created by calls to selected srs, buffered in CS()
    Two modes are possible-
        an external css file is created vs script in <head> of page
        External file may be shared.
